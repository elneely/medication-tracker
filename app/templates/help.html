Medication Questions

How do I add a medication?

You can add a medication by using the {% if user %}{% endif %} Add Medication{% if user %}{% endif %} link in the menu. The form allows you to enter a great deal of information about your medication - if it seems overwhelming, just remember that the only required information for a medication is its name. Anything else is optional!

So what information can you store about your medication? Click the sections below for more information on each part of the form.

  • Basic Information

    • Name: The only required field on this form, this is for the medication's name. Note that this does not have to be unique in case you have two different dosages of the same medication for some reason, say. However, if you have two medications with the same name you might want to note the difference between them somewhere.

    Brand name: Sometimes generic medication names are long and hard to spell or pronounce. If you have one of them, you might want to add a brand name since they are usually shorter and easier to remember. This is also helpful if the medical staff is inconsistent about which name they use; you won't be stuck trying to remember that Synthroid is Levothyroxine.

    Dose: What is the dosage of your medication? e.g., 500 mg, .75 mcg, 5mL, 2 teaspoons, etc.

    Frequency: How often do you need to take this medication? e.g., Twice a day, Once at bedtime, Once every 7 days, etc.

    Length: How long is this prescription? If you are on a long-term medication (sometimes called a maintenance medication) this could be 90 days. If not, it can range anywhere from 30 days to 1 day (although you probably don't need to enter it in the medication tracker if you only take it for a single day, unless you really want to!)

    Short-term medication?: On your user page your medications are divided into short-term and long-term medications. This checkbox allows you to specify a medication is short-term. If it is not checked, the medication is assumed to be long-term.

    Reason: It is easy to lose track of what medication you are taking for what purpose. This allows you to make a little note to remind yourself of why you are taking this medication.

  • Adding a Doctor or Pharmacy

    • Add Doctor: This allows you to record what doctor has prescribed your medication, which can be handy to know if you have run out of refills! You can either select a doctor you have previously used or you can add a new doctor. New doctors will automatically be added to your doctor contact list. If you change your mind about adding a doctor, just press the Don't Add Doctor button!

    Add Pharmacy: This allows you to keep track of what pharmacy you filled your prescription at. Just as for adding a doctor, you can select a pharmacy you have already used or you can add a new one; new ones will be added to your pharmacy contact list. If you change your mind about adding a pharmacy, press the Don't Add Pharmacy button and it will go away!

  • Details

    • Prescription Date: This is the date your doctor prescribed your medication. Not every medication label includes this information, but you can record it here if they do.

    Last Filled: This helps you keep track of when you last filled this prescription. Note that if you don't keep this up to date your refill reminders will stop being helpful!

    Prescription Expiration Date: Prescriptions don't last forever, so this helps you keep track of when your prescription or refills expire. Sometimes your medication label will also include when the medication itself expires (i.e., when it is past the date you should be using it.) For most of us the expiration date of the prescription is more useful to keep track of, since you can't refill a prescription that has expired.

    Number of refills remaining: A crucial piece of information for long-term medications, here you can keep track of how many refills you have left.

    Refill reminders on?: If you would like to be reminded of when you need to refill this medication, simply check this box and enter how many days in advance you wish to be reminded. (3 days before you run out? 7 days?) If you do wish to be reminded, you will also need to enter when the medication was last filled and how long the prescription is for. e.g., if your prescription is 30 days long and you want to be reminded 4 days before you run out, then your reminders will start 26 days after you last filled it.

    Notes: This is for anything else you wish to record about this medication. Do you tend to have side-effects from it? Do you need to take it with food or wait 30 minutes before you eat? Anything is fair game for this field!

How do I see detailed information for one of my medications?

While a summary of your medications appears on your {% if user %}{% endif %}home page{%- if user -%} {%- endif -%}, you can see all of the details for it by clicking on its name either there or on the {% if user %}{% endif %} Manage Medications{% if user %}{% endif %} page.

How do I edit the information for one of my medications?

Go to the page for the specific medication you wish to edit by clicking on its link either on your {% if user %}{% endif %}home page {%- if user -%}{% endif %} or on the {% if user %}{% endif %} Manage Medications{% if user %}{% endif %} page. Press the "Edit Information" button and you will be able to edit the information for this medication. For information on any of the form fields, please consult the information under "How do I add a medication?"

Note: the edit medication page only allows you to choose a currently existing doctor or pharmacy, not add a new one. If you have changed to a new doctor or pharmacy, please add them first (via the {% if user %}{% endif %}Add Doctor {%- if user -%}{% endif %} or {% if user %}{% endif %}Add Pharmacy {%- if user -%}{% endif %} pages) and then edit your medication.

You can also add a doctor or pharmacy to multiple medications at once via the {% if user %}{% endif %} Manage Medications{% if user %}{% endif %} page.

How do I delete a medication?

Go to the page for the specific medication you wish to edit by clicking on its link either on your {% if user %}{% endif %}home page {%- if user -%}{% endif %} or on the {% if user %}{% endif %} Manage Medications{% if user %}{% endif %} page. Press the "Delete Medication" button and a confirmation will appear; if you definitely want to delete the medication just confirm and press the delete button.

You can also delete multiple medications at once via the {% if user %}{% endif %} Manage Medications{% if user %}{% endif %} page.

How do I use the Manage Medications page?

The {% if user %}{% endif %} Manage Medications{% if user %}{% endif %} page allows you to perform certain actions on multiple medications at once. By checking the boxes next to the medications you wish to modify you can choose to change the doctor, change the pharmacy, or even delete the medications. (If you choose the "Delete Medications" option you will need to confirm your choice before they are deleted.) You can select or deselect all medications by checking the top box.

Note: this page only allows you to choose a currently existing doctor or pharmacy, not add a new one. If you have changed to a new doctor or pharmacy, please add them first (via the {% if user %}{% endif %}Add Doctor{%- if user -%} {% endif %} or {% if user %}{% endif %}Add Pharmacy {%- if user -%}{% endif %} pages) and then apply them to your medications.

How do I set and use refill reminders?

Refill reminders can be set either when you add a medication or when you edit it. If you would like to be reminded of when you need to refill a particular medication, simply check the appropriate box on the form and enter how many days in advance you wish to be reminded. You will also need to enter when the medication was last filled and how long the prescription is for. Refill reminders will appear on your {% if user %}{% endif %}home page{%- if user -%} {% endif %} at the appropriate times.

While refill reminders can only be changed from the information page for a specific medication, you can view all of your refill reminders on your {% if user %}{% endif %} profile page{% if user %}{% endif %}.

Doctor Questions

How do I add a doctor?

You can add a doctor by using the {% if user %}{% endif %} Add Doctor{% if user %}{% endif %} link in the menu or the "Add New Doctor" button on your {% if user %}{% endif %} doctor contact list {%- if user -%}{% endif %}.

You can store basic contact information for your doctor, along with any notes that you have (whether for remembering that this is your dentist or reminding yourself that there is no parking by their office - whatever you might find useful!) The only required field is "Last name," although each of your doctors is required to have a unique name, so if you have two with the same last name you might wish to provide a first name or some other notation to distinguish them. (If you have two doctors with the same first and last names, you will need to add some notation to distinguish them since otherwise there is no way to keep track of which one is which!)

How do I see detailed information for one of my doctors?

While a summary of the contact information for all of your doctors appears on your {% if user %}{% endif %}doctor contact list {%- if user -%}{% endif %}, you can see detailed information on a specific doctor simply by clicking on the doctor's name.

How do I edit the information for one of my doctors?

Go to the page for the specific doctor you wish to edit by clicking on its link on your {% if user %}{% endif %} doctor contact list{% if user %}{% endif %}. Press the "Edit Information" button and you will be able to edit the information for this doctor. For information on any of the form fields, please consult the information under "How do I add a doctor?"

How do I delete a doctor?

Go to the page for the specific doctor you wish to edit by clicking on its link on your {% if user %}{% endif %} doctor contact list{%- if user -%}{% endif %}. Press the "Delete Doctor" button and a confirmation will appear; if you definitely want to delete the doctor just confirm and press the delete button.

If you still have the doctor listed on any of your medications, don't worry - the medications will automatically be updated.

How do I view the contact page for all of my doctors?

The contact page for your doctors can be viewed by using the {% if user %}{% endif %}View Doctors {%- if user -%}{% endif %} link in the side menu.
